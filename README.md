- Some syntax
	Pulse i/p voltage

	Vname N1 N2 PULSE(V1 V2 TD Tr Tf PW Period)
	V1 - initial voltage; V2 - peak voltage; TD - initial delay time; Tr - rise time; Tf - fall time; pwf - pulse-wise; and Period - period.
